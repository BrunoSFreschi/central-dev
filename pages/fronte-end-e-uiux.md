Front-end (HTML, CSS, JavaScript, Frameworks)

[Frontend Mentor](https://www.frontendmentor.io/)
Desafios reais de Front-end baseados em layouts profissionais. Excelente para HTML, CSS, JS e frameworks modernos.

[FreeCodeCamp](https://www.freecodecamp.org/)
Plataforma gratuita com trilhas completas de Front-end, projetos práticos e certificações.

[JavaScript30](https://javascript30.com/)
Curso gratuito com 30 projetos em JavaScript puro, focado em prática e fundamentos reais.

[The Odin Project](https://www.theodinproject.com/)
Currículo completo de Front-end e Full Stack, com projetos progressivos e foco em boas práticas.

[CSS Challenges (Frontend Practice)](https://www.frontendpractice.com/)
Coleção de desafios baseados em sites reais, com foco em HTML e CSS modernos.

UI / UX Design (Design, Usabilidade, Produto)

[UX Challenge](https://uxtools.co/challenges)
Desafios semanais de UX para praticar pesquisa, ideação e solução de problemas reais.

[Sharpen.design](https://sharpen.design/)
Gerador de desafios aleatórios de UI/UX para praticar design de produto e pensamento criativo.

[Daily UI](https://www.dailyui.co/)
Desafios diários de UI para criar telas e interfaces, ótimo para quem usa Figma.

[UX Project Ideas (GitHub)](https://github.com/AdityaJoshi/UX-Design-Resources)
Coleção de recursos, ideias de projetos e estudos de caso em UX/UI.

[Designercize](https://designercize.com/)
Exercícios práticos de UX focados em pensamento de produto, não apenas estética.

Integração Front-end + UI/UX (Design → Código)

[Frontend Challenges (Codewell)](https://www.codewell.cc/)
Desafios de UI com foco em converter design em código fiel.

[Figma Community](https://www.figma.com/community)
Layouts reais, wireframes e sistemas de design gratuitos para estudo e implementação em código.

[UI Patterns](https://ui-patterns.com/)
Biblioteca de padrões de UI e UX usados em produtos reais.

[Open Source Design](https://www.opensourcedesign.net/)
Projetos open source que precisam de designers e front-ends — experiência prática de mercado.
