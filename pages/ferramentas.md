Editores, IDEs e Ambientes

[Visual Studio Code](https://code.visualstudio.com/)
Editor leve e extensível, padrão de mercado para Front-end, com vasto ecossistema de extensões.

[CodeSandbox](https://codesandbox.io/)
Ambiente online para prototipar, testar e compartilhar projetos Front-end rapidamente.

[StackBlitz](https://stackblitz.com/)
IDE online com foco em projetos JavaScript modernos (Angular, React, Vue) com execução instantânea.

[Replit](https://replit.com/)
Ambiente de desenvolvimento colaborativo para múltiplas linguagens, útil para protótipos rápidos.

---

Controle de Versão e Colaboração

[Git](https://git-scm.com/)
Sistema de controle de versão essencial para qualquer desenvolvedor profissional.

[GitHub](https://github.com/)
Hospedagem de código, colaboração, issues, pull requests e portfólio público.

[GitHub Desktop](https://desktop.github.com/)
Interface gráfica para Git, ideal para quem está aprendendo ou prefere visualização.

[GitLab](https://gitlab.com/)
Alternativa completa ao GitHub, com CI/CD integrado.

---

Debug, Inspeção e Performance

[Chrome DevTools](https://developer.chrome.com/docs/devtools/)
Ferramenta indispensável para inspecionar DOM, CSS, performance, network e acessibilidade.

[Lighthouse](https://developer.chrome.com/docs/lighthouse/)
Auditoria automática de performance, acessibilidade, SEO e boas práticas.

[WebPageTest](https://www.webpagetest.org/)
Testes avançados de performance e carregamento real de páginas.

---

Qualidade de Código e Padronização

[ESLint](https://eslint.org/)
Análise estática para identificar erros e inconsistências em JavaScript/TypeScript.

[Prettier](https://prettier.io/)
Formatação automática de código para manter consistência no time.

[Stylelint](https://stylelint.io/)
Linter específico para CSS e preprocessadores.

[Husky](https://typicode.github.io/husky/)
Automação de hooks do Git para garantir qualidade antes de commits.

Testes

[Jest](https://jestjs.io/)
Framework de testes unitários amplamente usado no Front-end.

[Vitest](https://vitest.dev/)
Framework moderno de testes, rápido e integrado ao Vite.

[Cypress](https://www.cypress.io/)
Testes end-to-end focados na experiência real do usuário.

[Playwright](https://playwright.dev/)
Automação e testes cross-browser com alto nível de confiabilidade.

UI, Design e Prototipação

[Figma](https://www.figma.com/)
Ferramenta padrão de mercado para UI, UX, protótipos e design systems.

[Storybook](https://storybook.js.org/)
Ambiente isolado para desenvolver e documentar componentes de UI.

[Zeplin](https://zeplin.io/)
Facilita o handoff entre design e desenvolvimento.

[Font Awesome](https://fontawesome.com/)
Biblioteca de ícones amplamente usada em projetos Front-end.

Deploy, Hospedagem e Preview

[Vercel](https://vercel.com/)
Deploy automático para aplicações Front-end modernas.

[Netlify](https://www.netlify.com/)
Hospedagem estática com CI/CD integrado e previews de PR.

[GitHub Pages](https://pages.github.com/)
Hospedagem simples para portfólios e projetos estáticos.
