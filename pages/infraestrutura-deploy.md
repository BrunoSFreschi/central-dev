Hospedagem e Deploy Front-end (Gratuito)

[GitHub Pages](https://pages.github.com/)
Hospedagem gratuita para sites estáticos diretamente a partir de repositórios GitHub.

[Netlify (Free Tier)](https://www.netlify.com/)
Deploy automático, previews de PR e CDN incluídos no plano gratuito.

[Vercel (Free Tier)](https://vercel.com/)
Deploy contínuo para Front-end moderno, com funções serverless limitadas gratuitamente.

[Cloudflare Pages](https://pages.cloudflare.com/)
Hospedagem estática gratuita com CDN global e excelente performance.

---

CI/CD Gratuito

[GitHub Actions](https://github.com/features/actions)
Automação gratuita para repositórios públicos (e cota mensal para privados).

[GitLab CI/CD](https://docs.gitlab.com/ee/ci/)
Pipelines gratuitos com boa cota de execução.

---

Backend / Serverless Gratuito (Complementar)

[Vercel Functions](https://vercel.com/features/functions)
Funções serverless com limites gratuitos suficientes para projetos de estudo.

[Netlify Functions](https://www.netlify.com/products/functions/)
Serverless integrado ao deploy Front-end no plano free.

[AWS Lambda (Free Tier)](https://aws.amazon.com/free/)
Execuções gratuitas por 12 meses (limite mensal).

Bancos de Dados Gratuitos (para projetos)

[Supabase](https://supabase.com/)
PostgreSQL gratuito com autenticação e storage.

[Firebase](https://firebase.google.com/)
Realtime Database, Auth e Hosting gratuitos em pequena escala.

[PlanetScale](https://planetscale.com/)
MySQL serverless com plano gratuito limitado.

[Neon](https://neon.tech/)
PostgreSQL serverless com free tier moderno.

[UptimeRobot](https://uptimerobot.com/)
